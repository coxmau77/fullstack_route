### Nivel Básico: Fundamentos Inquebrantables

Este nivel es el pilar de tu aprendizaje. Aquí construirás los conocimientos esenciales de las tecnologías web del lado del cliente y del servidor sin depender de frameworks. Es el "lenguaje nativo" que todo desarrollador Full Stack debe dominar.

#### Carril Frontend (FE): Fundamentos Esenciales

* **FE-BAS-01\_HTML5\_Moderno: HTML moderno para la estructura web.**
    * **Descripción del Módulo:** En este módulo, aprenderás a estructurar contenido web de forma semántica y accesible utilizando las últimas características de HTML5, desde la organización básica de una página hasta la integración de formularios avanzados y elementos multimedia.
    * **FE-BAS-01-01\_Estructura\_Semantica:** Uso correcto de etiquetas HTML5 (header, nav, main, section, article, aside, footer).
        * **Clase 1: Introducción a HTML5 y Estructura Semántica**
            * **Descripción:** Conocerás qué es HTML5, su importancia actual y la estructura fundamental de un documento web. Aprenderás a usar etiquetas semánticas como `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<aside>` y `<footer>` para organizar tu contenido de manera lógica y significativa.
        * **Clase 2: Listas, Textos y Enlaces**
            * **Descripción:** Explorarás las diversas etiquetas para formatear texto (encabezados, párrafos, énfasis) y crear listas. Dominarás el uso de la etiqueta `<a>` para construir enlaces internos y externos, esenciales para la navegación web.
        * **Clase 3: Imágenes y Multimedia**
            * **Descripción:** Aprenderás a insertar imágenes utilizando la etiqueta `<img>`, incluyendo atributos para accesibilidad y tamaño. También te sumergirás en la incorporación de elementos multimedia como `audio` y `video`, y verás cómo hacer tus imágenes responsivas con `<picture>`.
        * **Clase 4: Formularios (Parte 1)**
            * **Descripción:** Se te introducirá al mundo de los formularios HTML, comprendiendo su propósito y la estructura básica con la etiqueta `<form>`. Conocerás los elementos fundamentales como `<input>`, `<label>`, `<textarea>`, `<select>` y `<button>`, y sus atributos clave.
        * **Clase 5: Formularios (Parte 2) y Atributos Nuevos**
            * **Descripción:** Profundizarás en los tipos de `<input>` más avanzados de HTML5 (como `date`, `number`, `email`). Aprenderás a usar `datalist` para sugerencias y a implementar validaciones nativas de HTML5 (`required`, `pattern`, `min`, `max`).
        * **Clase 6: Accesibilidad (ARIA)**
            * **Descripción:** Entenderás la importancia de la accesibilidad web y los principios POUR. Aprenderás a utilizar atributos ARIA (`aria-label`, `aria-describedby`, `role`) para mejorar la experiencia de usuarios con tecnologías de asistencia, haciendo tus sitios más inclusivos.
        * **Consigna/Actividad Final del Módulo:** Crear una página web completa sobre un tema de tu elección, integrando estructura semántica, un formulario de contacto accesible con validaciones y elementos multimedia.
        * **Logro/Hito:** **Dominio HTML Semántico** (Has construido una estructura web accesible y moderna).

* **FE-BAS-02\_CSS3\_Avanzado\_y\_Responsivo: Estilización y diseño adaptable.**
    * **Descripción del Módulo:** Este módulo te capacitará para estilizar tus páginas web de forma profesional y garantizar que se vean impecables en cualquier dispositivo. Dominarás Flexbox y CSS Grid, y aprenderás a crear diseños adaptables con Media Queries y Variables CSS.
    * **FE-BAS-02-01\_Selectores\_Propiedades\_Avanzadas:** Pseudoclases, pseudoelementos, transformaciones, transiciones, animaciones básicas.
        * **Clase 1: Selectores y Propiedades Avanzadas**
            * **Descripción:** Aprenderás a usar selectores CSS más complejos (atributos, pseudoclases como `:hover`, `:focus`, `:nth-child()`, y pseudoelementos como `::before`, `::after`) para un control de estilo preciso. También te introducirás a transformaciones, transiciones y animaciones básicas.
        * **Clase 2: Modelo de Caja y Contextos de Apilamiento**
            * **Descripción:** Profundizarás en el Modelo de Caja de CSS, comprendiendo `margin` collapsing y `box-sizing`. Dominarás la propiedad `position` (`relative`, `absolute`, `fixed`, `sticky`) y el manejo de capas con `z-index` en diferentes contextos de apilamiento.
        * **Clase 3: Flexbox (Parte 1)**
            * **Descripción:** Conocerás los fundamentos de Flexbox, una de las herramientas más poderosas para la creación de layouts flexibles. Aprenderás las propiedades del contenedor Flex (`display: flex`, `flex-direction`, `justify-content`, `align-items`, `flex-wrap`) para alinear y distribuir elementos.
        * **Clase 4: Flexbox (Parte 2)**
            * **Descripción:** Continuarás con Flexbox, enfocándote en las propiedades de los ítems Flex (`flex-grow`, `flex-shrink`, `flex-basis`, `order`, `align-self`). Aprenderás patrones de diseño comunes y cómo construir layouts complejos y responsivos usando Flexbox.
        * **Clase 5: CSS Grid Layout (Parte 1)**
            * **Descripción:** Te introducirás a CSS Grid, el sistema de diseño bidimensional que permite controlar tanto filas como columnas. Aprenderás las propiedades básicas del contenedor Grid (`display: grid`, `grid-template-columns`, `grid-template-rows`, `gap`).
        * **Clase 6: CSS Grid Layout (Parte 2)**
            * **Descripción:** Profundizarás en CSS Grid, explorando las propiedades de los ítems (`grid-column`, `grid-row`, `grid-area`) y la creación de layouts con áreas nombradas (`grid-template-areas`). También verás el auto-placement y la alineación avanzada.
        * **Clase 7: Diseño Responsivo (Mobile-First) y Variables CSS**
            * **Descripción:** Adoptarás la metodología "Mobile-First" para un diseño responsivo. Dominarás las Media Queries avanzadas y el uso de unidades relativas (`rem`, `em`, `vw`, `vh`). Además, aprenderás a usar Variables CSS (`Custom Properties`) para temas y reutilización de estilos.
        * **Consigna/Actividad Final del Módulo:** Re-estilizar la página web creada en el módulo de HTML para que sea completamente responsiva, utilizando las técnicas de Flexbox, CSS Grid y Media Queries, e implementando Variables CSS para una personalización sencilla.
        * **Logro/Hito:** **Maestro CSS Responsivo** (Has logrado que tus diseños se vean bien en cualquier dispositivo).

* **FE-BAS-03\_JavaScript\_Moderno\_ES\_Ultimas\_Versiones: El lenguaje del navegador y la base de los frameworks.**
    * **Descripción del Módulo:** Este es el módulo central del Frontend. Aprenderás JavaScript desde cero, cubriendo las características modernas de ES6+ (ECMAScript 2015 en adelante), el manejo de la asincronía, la manipulación del DOM y cómo interactuar con APIs externas.
    * **FE-BAS-03-01\_Variables\_Let\_Const\_Hoisting:** `let`, `const` vs `var`, ámbito de variables.
        * **Clase 1: Variables, Tipos de Datos y Operadores**
            * **Descripción:** Aprenderás sobre los diferentes tipos de datos en JavaScript (primitivos y objetos), cómo declarar variables con `let` y `const` (y las diferencias con `var`), y el uso de operadores aritméticos, de comparación y lógicos.
        * **Clase 2: Estructuras de Control y Funciones Básicas**
            * **Descripción:** Dominarás las estructuras de control de flujo (`if/else`, `switch`, `for`, `while`) para manejar la lógica de tus programas. Te introducirás a la creación y uso de funciones, un pilar de la programación en JavaScript.
        * **Clase 3: Funciones Flecha, Contexto `this` y Programación Funcional**
            * **Descripción:** Conocerás las concisas Funciones Flecha (`=>`) y su impacto en el contexto de `this`. Además, te introducirás a conceptos de programación funcional usando métodos de array esenciales como `map`, `filter` y `reduce`.
        * **Clase 4: Desestructuración, Spread y Rest Operators**
            * **Descripción:** Aprenderás técnicas modernas para manejar arrays y objetos: la desestructuración para extraer valores convenientemente, el operador `...spread` para copiar o combinar elementos, y el operador `...rest` para agrupar parámetros de función.
        * **Clase 5: Módulos ES Modules (`import`/`export`)**
            * **Descripción:** Entenderás la importancia de modularizar tu código JavaScript para una mejor organización y mantenimiento. Aprenderás a usar la sintaxis `import` y `export` de ES Modules para compartir funcionalidades entre archivos.
        * **Clase 6: Asincronía (Callbacks, Promises)**
            * **Descripción:** Te introducirás al concepto crucial de la asincronía en JavaScript. Aprenderás sobre los Callbacks (y sus desafíos como el "callback hell") y cómo las Promises ofrecen una solución más limpia para manejar operaciones asíncronas.
        * **Clase 7: Asincronía (`async/await`) y Fetch API**
            * **Descripción:** Dominarás la sintaxis `async/await`, la forma más moderna y legible de trabajar con código asíncrono, que simplifica el uso de Promises. También aprenderás a usar `Fetch API` para realizar peticiones HTTP a servidores y obtener datos.
        * **Clase 8: Manipulación del DOM Avanzada y Event Delegation**
            * **Descripción:** Aprenderás a interactuar dinámicamente con el Document Object Model (DOM) del navegador: seleccionar elementos, crear nuevos, modificarlos y eliminar. Conocerás el concepto de Event Delegation para manejar eventos de manera eficiente en elementos dinámicos.
        * **Clase 9: Manejo de Errores y Depuración**
            * **Descripción:** Entenderás los tipos comunes de errores en JavaScript y cómo manejarlos usando bloques `try/catch`. Te familiarizarás con las herramientas de desarrollo del navegador para depurar y resolver problemas en tu código.
        * **Consigna/Actividad Final del Módulo:** Crear una aplicación web simple (ej. una lista de tareas persistente en el navegador, o un buscador de películas básico) utilizando solo HTML, CSS y JavaScript moderno, incluyendo la interacción con una API pública.
        * **Logro/Hito:** **Cinturón Negro JS** (Has construido una aplicación web interactiva y asíncrona desde cero).

#### Carril Backend (BE): Fundamentos de Node.js

* **BE-BAS-01\_NodeJS\_Fundamentos:** El entorno de ejecución de JavaScript en el servidor.
    * **Descripción del Módulo:** Este módulo te introduce a Node.js, el entorno que permite a JavaScript ejecutarse fuera del navegador. Aprenderás a crear servidores web básicos, a gestionar paquetes con NPM y a entender el modelo de asincronía de Node.js.
    * **BE-BAS-01-01\_Introduccion\_Node\_NPM:** Qué es Node.js, gestor de paquetes NPM, `package.json`.
        * **Clase 1: Introducción a Node.js y NPM**
            * **Descripción:** Conocerás qué es Node.js, cómo difiere de JavaScript en el navegador, y su rol en el desarrollo Backend. Aprenderás a usar NPM (Node Package Manager) para gestionar dependencias de proyectos y entender el archivo `package.json`.
        * **Clase 2: Módulos Comunes y `http` (Servidor Básico)**
            * **Descripción:** Explorarás módulos Built-in de Node.js como `fs` (File System) para interactuar con archivos, y `path` para manejar rutas. Construirás tu primer servidor web básico utilizando el módulo `http` de Node.js.
        * **Clase 3: El Event Loop y Asincronía en Node.js**
            * **Descripción:** Este es un tema crucial: comprenderás a fondo el Event Loop de Node.js, el mecanismo que le permite manejar operaciones de E/S de forma no bloqueante. Verás cómo interactúan `setTimeout`, `setInterval` y `process.nextTick`.
        * **Clase 4: Manejo de Eventos (`EventEmitter`) y Streams**
            * **Descripción:** Aprenderás sobre el patrón de diseño `EventEmitter` para crear y escuchar eventos personalizados en tus aplicaciones Node.js. Te introducirás a los `Streams` para manejar eficientemente grandes volúmenes de datos.
        * **Consigna/Actividad Final del Módulo:** Crear un servidor HTTP básico con Node.js que sea capaz de servir archivos estáticos (como HTML, CSS, JS) y también una API muy simple que devuelva datos JSON estáticos desde un archivo.
        * **Logro/Hito:** **Conquistador del Servidor** (Has levantado tu primer servidor y entendido el Event Loop de Node.js).

* **BE-BAS-02\_ExpressJS\_Introduccion:** El framework más popular para construir APIs REST en Node.js.
    * **Descripción del Módulo:** En este módulo, te sumergirás en Express.js, el framework web más popular para Node.js. Aprenderás a construir APIs RESTful de manera eficiente, gestionando rutas, peticiones, respuestas HTTP, y manejando errores.
    * **BE-BAS-02-01\_Conceptos\_Frameworks\_Backend:** Por qué usar un framework como Express.js.
        * **Clase 1: ¿Por qué Express.js? Y Configuración Inicial**
            * **Descripción:** Entenderás las ventajas de usar un framework como Express.js sobre el módulo `http` nativo de Node.js. Aprenderás a instalar y configurar una aplicación Express básica, sirviendo tu primera respuesta.
        * **Clase 2: Rutas y Métodos HTTP**
            * **Descripción:** Dominarás la definición de rutas en Express.js para diferentes métodos HTTP (`GET`, `POST`, `PUT`, `DELETE`). Aprenderás a extraer datos de la URL usando `req.params` (parámetros de ruta) y de la cadena de consulta con `req.query`.
        * **Clase 3: Middlewares y Cuerpo de la Petición (`req.body`)**
            * **Descripción:** Descubrirás el concepto fundamental de los Middlewares en Express.js y cómo usarlos para interceptar y modificar peticiones. Aprenderás a parsear el cuerpo de las peticiones (`req.body`) para recibir datos enviados desde el cliente (ej. formularios o JSON).
        * **Clase 4: Manejo de Errores y Estructura del Proyecto**
            * **Descripción:** Aprenderás a implementar un middleware de manejo de errores global en Express.js para capturar y responder a errores de manera consistente. También verás patrones básicos para estructurar tu proyecto Express, separando rutas y lógica.
        * **Consigna/Actividad Final del Módulo:** Refactorizar el servidor Node.js creado en el módulo anterior en una API REST completa utilizando Express.js, implementando las operaciones CRUD (Crear, Leer, Actualizar, Borrar) para una colección de recursos (ej. usuarios, productos).
        * **Logro/Hito:** **Constructor de APIs** (Has creado tu primera API REST funcional con Express.js).

### Nivel Intermedio: Consolidación y Conexión

Este nivel actúa como un puente entre los fundamentos y las especializaciones avanzadas. Aquí aplicarás tus conocimientos básicos en un proyecto integrado, comprendiendo cómo Frontend y Backend interactúan, y abordando temas comunes en aplicaciones reales.

#### Pre-requisitos Obligatorios:

* Completar todos los módulos de **FE-BAS** (HTML, CSS, JS).
* Completar todos los módulos de **BE-BAS** (Node.js, Express.js).

#### Carril Frontend (FE): Aplicación Práctica con API Simulado

* **FE-INT-01\_Consolidacion\_FE\_Proyecto:** Construcción de un Frontend dinámico con JS puro.
    * **Descripción del Módulo:** Este módulo te desafiará a construir aplicaciones Frontend más complejas utilizando únicamente JavaScript puro. Te enfocarás en la organización del código a gran escala, la creación de componentes reutilizables y el manejo avanzado de la asincronía y la interacción con el DOM.
    * **FE-INT-01-01\_Organizacion\_Codigo\_JS:** Patrones de diseño JavaScript (Módulos, IIFE, Factory Functions) para proyectos medianos.
        * **Clase 1: Organización de Código JS a Gran Escala**
            * **Descripción:** Aprenderás a estructurar proyectos JavaScript más grandes sin frameworks. Explorarás patrones de diseño como Módulos ES, IIFE (Immediately Invoked Function Expressions) y Factory Functions para mejorar la organización, la encapsulación y la reutilización de código.
        * **Clase 2: Creación de Componentes Reutilizables (JS puro)**
            * **Descripción:** Descubrirás cómo emular la idea de "componentes" que se ven en los frameworks, utilizando funciones y manipulación del DOM en JavaScript puro. Aprenderás a crear elementos de interfaz reutilizables con su propia lógica y estado.
        * **Clase 3: Interacción Compleja del DOM y Event Delegation Avanzada**
            * **Descripción:** Profundizarás en técnicas avanzadas de manipulación del DOM para construir interfaces altamente interactivas. Dominarás Event Delegation para manejar eventos de manera eficiente, especialmente en listas o elementos generados dinámicamente.
        * **Clase 4: Manejo de Asincronía Avanzada con Múltiples Peticiones**
            * **Descripción:** Aprenderás a orquestar múltiples peticiones asíncronas utilizando `Promise.all()` y `Promise.race()`. Dominarás estrategias para gestionar las dependencias y el flujo de datos cuando necesitas información de varias fuentes simultáneamente.
        * **Consigna/Actividad Final del Módulo:** Construir un proyecto interactivo más complejo (ej. una aplicación de gestión de citas con un calendario dinámico, o un buscador de contenido con múltiples filtros y paginación) utilizando **solo HTML, CSS y JS puro**, que consuma datos de varias APIs públicas o de una API simulada.
        * **Logro/Hito:** **Arquitecto Frontend Básico** (Has diseñado y construido una aplicación interactiva Frontend compleja sin frameworks).

#### Carril Backend (BE): Conexión con Base de Datos NoSQL

* **BE-INT-01\_Conexion\_Base\_Datos\_NoSQL:** Introducción a la persistencia de datos real.
    * **Descripción del Módulo:** Este módulo te introducirá al mundo de las bases de datos NoSQL, centrándose en MongoDB. Aprenderás a modelar datos orientados a documentos, a realizar operaciones CRUD y a integrar MongoDB con tu aplicación Node.js usando Mongoose.
    * **BE-INT-01-01\_Fundamentos\_Bases\_Datos:** Conceptos de Bases de Datos Relacionales vs. No Relacionales.
        * **Clase 1: Conceptos de Bases de Datos y SQL vs NoSQL**
            * **Descripción:** Conocerás los conceptos fundamentales de las bases de datos. Explorarás las diferencias clave entre las bases de datos Relacionales (SQL) y No Relacionales (NoSQL), entendiendo cuándo es apropiado usar cada tipo y sus ventajas.
        * **Clase 2: Introducción a MongoDB**
            * **Descripción:** Te sumergirás en MongoDB, una popular base de datos NoSQL orientada a documentos. Aprenderás su modelo de datos (colecciones y documentos), cómo instalarla localmente o usar un servicio en la nube como MongoDB Atlas, y a interactuar con su interfaz básica.
        * **Clase 3: Operaciones CRUD con MongoDB Shell y Mongoose (Parte 1)**
            * **Descripción:** Practicarás las operaciones CRUD (Crear, Leer, Actualizar, Borrar) directamente desde el shell de MongoDB. Luego, te introducirás a Mongoose, una librería ODM (Object Data Modeling) para Node.js que simplifica la interacción con MongoDB mediante esquemas y modelos.
        * **Clase 4: Implementación CRUD en Express.js con Mongoose (Parte 2)**
            * **Descripción:** Conectarás tu aplicación Express.js con MongoDB utilizando Mongoose. Implementarás rutas API completas para realizar operaciones CRUD en tus colecciones de datos (ej. "productos", "usuarios"), persistiendo y recuperando información real de la base de datos.
        * **Consigna/Actividad Final del Módulo:** Crear una API REST completa en Express.js que gestione una colección de datos (ej. usuarios, posts de blog) y persista esa información en una base de datos MongoDB.
        * **Logro/Hito:** **Guardián de Datos** (Has conectado tu API a una base de datos real y manejas la persistencia de información).

* **PROJ-01\_App\_Basica\_FullStack:** Integración de Frontend y Backend.
    * **Descripción del Módulo:** Este es tu primer proyecto Full Stack real. Integrarás el Frontend que creaste con JavaScript puro con la API Backend que desarrollaste con Node.js y Express.js, y aprenderás a desplegar ambas partes en servicios de hosting.
    * **PROJ-01-01\_Conexion\_FE\_BE\_API\_Propia:** Conectar el Frontend de un proyecto básico (ej. tu lista de tareas o buscador) con la API Backend que creaste en **BE-INT-01**.
        * **Clase 1: Planificación y Diseño de la Integración**
            * **Descripción:** Revisarás tus proyectos de Frontend (JS puro) y Backend (API con MongoDB). Planificarás cómo se comunicarán, definiendo los endpoints necesarios y el flujo de datos entre el cliente y el servidor para una funcionalidad clave (ej. añadir un elemento al Backend desde el Frontend).

        * **Logro/Hito Principal:** **Pionero Full Stack** (Has construido y desplegado tu primera aplicación Full Stack completa, conectando ambos mundos).

* **Ruta: FE-ADV-ANG\_Angular\_Profesional:** Desarrollo Frontend con el framework Angular.
    * **Descripción de la Ruta:** Esta ruta te sumergirá en el ecosistema de Angular, uno de los frameworks más completos para construir Single Page Applications (SPAs) empresariales. Aprenderás su arquitectura basada en componentes, la inyección de dependencias, la gestión de estado, el enrutamiento y las mejores prácticas de desarrollo y testing.
    * **Pre-requisitos específicos:** **FE-BAS-01\_HTML5\_Moderno**, **FE-BAS-02\_CSS3\_Avanzado\_y\_Responsivo**, **FE-BAS-03\_JavaScript\_Moderno\_ES\_Ultimas\_Versiones**, **FE-INT-01\_Consolidacion\_FE\_Proyecto**.

        * **Clase 1:** Introducción al Ecosistema Angular y CLI
            * **Descripción:** Conocerás qué es Angular, su filosofía y cómo usar Angular CLI para iniciar y gestionar proyectos. Crearás tu primer proyecto y explorarás su estructura.
        * **Clase 2:** Componentes y Data Binding
            * **Descripción:** Dominarás el concepto de componentes en Angular, su rol como bloques de construcción. Aprenderás las diferentes formas de "Data Binding" (interpolación, property binding, event binding, two-way binding) para conectar la lógica con la UI.
        * **Clase 3:** Directivas y Pipes
            * **Descripción:** Explorarás las Directivas (`*ngIf`, `*ngFor`, `[ngClass]`, `[ngStyle]`) para manipular el DOM dinámicamente y los Pipes para transformar datos directamente en la plantilla.
    * **FE-ADV-ANG-02\_Servicios\_Inyeccion\_Dependencias:** Gestión de lógica, servicios Singleton.
        * **Clase 1:** Creación y Uso de Servicios
            * **Descripción:** Aprenderás a crear Servicios en Angular para encapsular lógica de negocio y compartirlos entre componentes. Entenderás cómo se usan para separar responsabilidades y hacer tu código más modular.
        * **Clase 2:** Inyección de Dependencias
            * **Descripción:** Comprenderás el patrón de Inyección de Dependencias de Angular, un pilar del framework. Verás cómo los servicios son provistos y consumidos en componentes y otros servicios.
    * **FE-ADV-ANG-03\_Enrutamiento\_Avanzado\_Angular:** Lazy Loading, Guards.
        * **Clase 1:** Configuración de Rutas y Navegación
            * **Descripción:** Aprenderás a configurar el módulo de enrutamiento de Angular para crear Single Page Applications. Dominarás la navegación programática y declarativa, y el uso de parámetros de ruta.
        * **Clase 2:** Lazy Loading y Guards
            * **Descripción:** Optimizarás el rendimiento de tu aplicación con Lazy Loading de módulos, cargando código solo cuando es necesario. Implementarás Guards (`CanActivate`, `CanDeactivate`) para proteger rutas y controlar el acceso de usuarios.
